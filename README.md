# Tim Gray · 7 Days to Die Fansite

[![YouTube](https://img.shields.io/badge/YouTube-@Tim__Gray-red?style=flat&logo=youtube)](https://www.youtube.com/@Tim_Gray)

Фан-сайт для канала **Tim Gray**, посвящённый игре *7 Days to Die*.  
Содержит подробную информацию об игре, бестиарий зомби, гайды для новичков, обзор оружия и советы по выживанию.

## 📦 Содержание

- 🧟 Бестиарий (все типы зомби)
- 🛠️ Гайды для начинающих
- 🔫 Лучшее оружие
- 💡 Советы от Tim Gray
- 🔗 Прямые ссылки на YouTube канал

## 🚀 Как запустить

1. Клонируй репозиторий или скачай архив.
2. Открой файл `index.html` в любом современном браузере.
3. (Рекомендуется) Используй локальный сервер, например расширение **Live Server** для VSCode.

## 🛠️ Технологии

- HTML5
- CSS3 (Tailwind CSS + кастомные стили)
- JavaScript (ванильный, для интерактивности)
- Font Awesome 6 (иконки)


## 📺 О канале Tim Gray

Tim Gray — русскоязычный ютубер, создающий контент по игре **7 Days to Die** (версии 1.0+ и 2.5/2.6).  
На канале выходят гайды, разборы обновлений, советы по выживанию, летсплеи и стримы.
